# Stack

This repository contains Java implementations of the Stack data structure using Linked Lists as part of the Bridgelabz Data Structures and Algorithms learning program.

## Overview

The project demonstrates the implementation of a Stack without using Java's built-in `Stack` class. The Stack follows the **LIFO (Last In, First Out)** principle and is implemented using a singly linked list.

## Features

- Stack implementation using Linked List
- Push operation
- Peek operation
- Pop operation
- Display stack elements
- Dynamic memory allocation using linked nodes

## Use Cases

### UC1 - Add Elements to the Stack

Implemented the Stack using a Linked List and inserted elements using the **Push** operation.

**Objectives**

- Create a Stack using Linked List.
- Insert elements at the top of the Stack.
- Display the Stack after insertion.

---

### UC2 - Peek and Pop Operations

Implemented **Peek** and **Pop** operations on the Stack.

**Objectives**

- Retrieve the top element using the Peek operation.
- Remove the top element using the Pop operation.
- Display the updated Stack after deletion.
